# FileSorter

## What this does
FileSorter, as the name suggests sorts files (from the `Download` folder) into folder(s) spesified in `config.json`.

## How to use

*in the folder where `logic.py` is located* run `python logic.py`.

## How to configure

in `config.json`, write something like ***example***
```json
"FolderName/Path/If/Needed"[
    ".ext"
]
```
